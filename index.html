Universal Translator

Bridging language barriers through real-time AI-powered communication — speak your language, and let the app do the rest.

Universal Translator is a minimal and powerful voice translation app that enables two people to converse in different languages in real time. Built using Google Cloud’s Speech-to-Text, Translation, and Text-to-Speech APIs, it demonstrates strong grasp over AI-driven language workflows and edge-ready system design.

Key Features

  • 🎙️ Dual mic interface — "You" and "Them" with language selection
  • 🌍 Supports 100+ languages with native audio playback
  • 🔁 Auto-plays translated speech instantly
  • 🎧 Stores audio in Google Cloud Storage for 30 days
  • 🧠 Smart gender-aware voice synthesis (AI-generated)
  • 📊 Signal-to-noise quality detection and fallbacks
  • 🛡️ Firebase Auth (configurable) + secure endpoints
